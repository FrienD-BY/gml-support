
# Basic GameMaker Language support


## Features
- Full syntax highlight
- JSDoc comments 
- Regions
- Simple indentation

## Installation

with [Lazy](https://github.com/folke/lazy.nvim):

```lua
{
    "FrienD-BY/gml-support",
    event = { "BufReadPost", "BufNewFile" },
}
```
